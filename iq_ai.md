# IQ Ecosystem Overview

## IQ AI

**Reward:** `$700`

**IQ AI** is a **DeFAI-focused platform** that enables the creation and deployment of **autonomous, tokenized AI agents on-chain**.

It provides an **open-source TypeScript Agent Development Kit (ADK)** that allows developers to build:

- Simple automated bots  
- Autonomous on-chain AI agents  
- Complex **multi-agent systems**

This toolkit enables developers to design intelligent agents that can interact with blockchain protocols, automate workflows, and execute decentralized financial strategies.

## KRWQ

**KRWQ** launched in **October 2025** as the **first fully fiat-backed Korean Won (KRW) stablecoin**, developed by **IQ** in collaboration with **Frax Finance**.

### Key Highlights

- Fully **fiat-backed KRW stablecoin**
- Native **cross-chain functionality via LayerZero**
- Initially launched on **Base**
- Expanded support to **Ethereum**
- Achieved **over ₩4B in trading volume** by the end of 2025

KRWQ aims to improve **cross-chain liquidity and accessibility** for Korean Won-denominated digital assets in the decentralized finance ecosystem.
